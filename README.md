# Operation Mühle

Programmierprojekt im dritten Semester WS12/13, Gruppe 2

# Starten

## Linux und OS X

    cd export
    ./operation_muehle.sh

## Windows

    cd export
    java -jar operation_muehle.jar

# AIs laden

Um die AIs verfügbar zu machen reicht es das `jar` in das Verzeichniss lib/
zu tun, siehe mitgelieferte Beispiele.
